# Air Battle v2 (Pygame Project)

This is my first complete playable project built with Python and Pygame.

Compared to my previous version, this project focuses on improving **code structure and overall completeness**:

* Refactored from a single-file script into a **multi-file project structure** (main / entities / systems / settings)
* Added a **basic upgrade system** to enhance gameplay progression
* Built a full **game loop flow** (start → playing → game over / win → restart)
* Introduced simple **UI elements** (HP, score, level, boss HP)
* Integrated image assets and packaged into a **playable ZIP version**

This project marks the transition from “just making something run” to building a **structured and complete project**.
